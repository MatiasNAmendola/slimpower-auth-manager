Authentication Manager for SlimPower Framework

## Installation

In terminal:

```bash
composer require matiasnamendola/slimpower-auth-manager
```

Or you can add use this as your composer.json:

```json
{
    "require": {
        "matiasnamendola/slimpower-auth-manager": "dev-master"
    }
}
```
